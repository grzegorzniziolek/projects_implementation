House Price Prediction

This project is a machine learning model that predicts house prices based on various features such as location, size, number of bedrooms, and more. The model is trained on a dataset of historical house prices and can be used to estimate the value of a property given its characteristics.

The project is implemented using Python and Flask for the web application, and scikit-learn for the machine learning model. The model is trained using a dataset of house prices and their corresponding features, and then saved as a pickle file for later use in the web application.
